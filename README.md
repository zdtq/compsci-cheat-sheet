# Cheat Sheet for *Algoritmi e Principi dell'Informatica* @ PoliMi

This cheat sheet is designed for the course [*Algoritmi e Principi dell'Informatica*](https://onlineservices.polimi.it/manifesti/manifesti/controller/ManifestoPublic.do?EVN_DETTAGLIO_RIGA_MANIFESTO=evento&aa=2023&k_cf=225&k_corso_la=358&k_indir=I3I&codDescr=086067&lang=IT&semestre=2&idGruppo=4750&idRiga=296961) at Politecnico di Milano. The course combines key topics from **Theoretical Computer Science** (TCS) and **Data Structures & Algorithms** (DSA).

This document serves as a quick reference for essential concepts, definitions, and problem-solving techniques covered throughout the course.

**Format**: Optimized for printing on two regular A4 pages (front and back). Select 'Landscape' orientation when printing.

***\*NOTE\*** The cheat sheet has not been extensively revised, so there may be errors. If you find any, please contact me at the email listed on my profile.*

- Formal Languages
- Finite State Automata (FSA)
- Pumping Lemma for FSA
- Pushdown Automata (PDA)
- Turing Machines (TM)
- Random-Access Machines (RAM)
- First Order Monadic Logic (MFO)
- Second Order Monadic Logic (MSO)

### Data Structures & Algorithms

- Big O, Omega and Theta
- Uniform vs Logarithmic Cost Model
- Recurrences: Master Theorem & Substition 
- Arrays
- Lists
- Stacks
- Queues and Double-Ended Queues (Deques)
- Hash Tables & collision handling
- Trees: Binary Trees, Binary Search Trees (BST), Red-Black Trees (RB Trees), Binary Heaps 
- Graphs: Breadth-First Search (BFS), Depth-First Search (DFS), Connected Components, Topological Sorting, Shortest Path, Cycle Detection
- Sorting Algorithms: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Heap Sort, Counting Sort and complexity comparison
